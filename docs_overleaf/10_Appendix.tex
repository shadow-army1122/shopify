\chapter{APPENDIX}

\section{Minimum Software Requirements}
\medskip
\textbf{Operating System}: Windows 10/11, macOS, or Linux
\\\\
\textbf{Web Server / Environment}: PHP 8.2 or higher, Node.js (for asset compilation), Composer, and a relational database like MySQL or PostgreSQL.
\\\\
\textbf{Web Browser}: Any modern browser (Chrome, Firefox, Edge, Safari) for accessing the platform.

\section{Minimum Hardware Requirements}
\textbf{Processor}: Intel Core i5 (or AMD equivalent) or higher to efficiently run a local development server and compile Vite assets.
\\\\
\textbf{RAM}: 8 GB (16 GB is recommended for simultaneous backend and frontend development).
\\\\
\textbf{Storage}: 1 GB of free space for the application, vendor files, and node\_modules.
\\\\
\textbf{Display}: Standard HD resolution (1920x1080) for clear interface management and layout debugging.
\\\\
\textbf{Network Connectivity}: A stable internet connection for downloading Composer/NPM dependencies and generating embeddings via external AI models.
