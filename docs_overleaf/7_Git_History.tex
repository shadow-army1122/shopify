\chapter{GIT HISTORY}
To ensure proper version management and efficient collaboration, the Shopify project leverages Git for version control. This allows for smooth tracking of source code updates, database migrations, frontend UI refinements, and feature branches. The repository serves as a centralized location for maintaining the application, making it easier to roll back changes or track the injection of new features such as the integration of embeddings and seller dashboards.
