\addcontentsline{toc}{chapter}{ABSTRACT}  
\vspace{-2cm}  

Shopify is a comprehensive e-commerce platform built using the Laravel web framework combined with a modern React and Inertia.js frontend. The system is designed to provide a seamless shopping experience for both buyers and sellers, featuring a fully functional seller dashboard, product embedding algorithms for recommendations, robust authentication, and order tracking. 

The backend architecture leverages Laravel's Eloquent ORM to manage complex relational data for products, orders, and embeddings, while the frontend utilizes Tailwind CSS to deliver an aesthetically pleasing, responsive interface. By incorporating automated product embeddings and a dedicated seller module, this application serves as a robust foundation for an advanced Shopify operation, efficiently handling the dynamic needs of modern online retail.
